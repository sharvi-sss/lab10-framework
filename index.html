Welcome to My Static Web Page

This page demonstrates how I used a web framework/library for Lab 10.

Framework / Library Used

I chose MaterializeCSS, a lightweight front-end framework that simplifies styling and layout.

Installation & Setup

  • Created a GitHub repository for Lab 10
  • Cloned the repository locally
  • Added index.html and style.css
  • Included MaterializeCSS via CDN in the HTML <head> section
  • Pushed changes to GitHub
  • Enabled GitHub Pages in repository settings → chose main branch / root

Difficulties & Solutions

  • Initially forgot to include Materialize CDN → fixed by adding correct <link>
  • Text alignment and spacing were off → fixed with custom CSS
  • Images were not appearing → made sure they were in the correct folder
