Framework / Library Used

I chose MaterializeCSS, a lightweight front-end framework that makes styling easy and responsive.

Installation & Setup
  • Created a GitHub repository for Lab 10
  • Cloned it locally
  • Added index.html and style.css
  • Included MaterializeCSS via CDN in the HTML <head>
  • Pushed changes to GitHub
  • Enabled GitHub Pages in repository settings → main branch / root
Difficulties & Solutions

  • Text alignment and spacing looked off → fixed using Materialize utility classes and custom CSS
  • Images were not appearing → ensured correct folder and path
